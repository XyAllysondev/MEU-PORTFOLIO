# Portfólio — Allyson Vinícius

Portfólio pessoal desenvolvido do zero com **HTML5, CSS puro e JavaScript vanilla** — sem frameworks ou dependências externas.

🔗 **Ao vivo:** _(configure o GitHub Pages para publicar)_

## ✨ Destaques

- Design dark premium com aurora animada e constelação de partículas interativa
- Cursor customizado, botões magnéticos e spotlight nos cards
- Avatar com anéis rotativos, órbitas de partículas e glow pulsante
- Animações de scroll reveal, contadores e barras de skill
- Formulário de contato funcional (Formspree)
- Totalmente responsivo e com suporte a `prefers-reduced-motion`

## 🛠️ Tecnologias

- HTML5 semântico
- CSS3 (Grid, Flexbox, animações, backdrop-filter)
- JavaScript vanilla (IntersectionObserver, Canvas API)
